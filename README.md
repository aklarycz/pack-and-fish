# Paper Empire

HTML5 game prototype.

## Run

Open `index.html` in a browser — or, for hot reload and ES modules, use VS Code's **Live Server** extension (right-click `index.html` → "Open with Live Server").

## Structure

```
index.html       entry point
src/
  game.js        game logic (ES module)
  style.css      styling
assets/          images, audio, sprites
```
